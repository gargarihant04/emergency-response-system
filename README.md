# Real-Time Emergency Response System

A cloud-native, microservices-based emergency response platform designed to facilitate rapid SOS alerting and real-time location tracking.

## Architecture
* **Frontend:** [React Native / Flutter]
* **Backend:** [Node.js / Python]
* **Infrastructure:** Terraform (AWS Cloud)
* **Orchestration:** Docker & Kubernetes
* **CI/CD:** Jenkins

## Directory Structure
* `/frontend` - Mobile application UI
* `/backend` - Core API for routing alerts
* `/infrastructure` - IaC for AWS provisioning
* `/k8s` - Kubernetes deployment manifests
